CURSO: Sistemas para Internet


Informações sobre o curso:

O profissional de Sistemas para Internet é o responsável por “construir” todas as soluções conectadas que conhecemos.
Por meio de linguagens de programação e entendimentos específicos, é quem elabora os diversos recursos que usamos
todos os dias. Além dos aspectos estéticos, o web designer preocupa-se com a acessibilidade e usabilidade das páginas
na Internet. O desenvolvedor web utiliza códigos de programação para criar sites, portais, aplicativos, lojas virtuais,
fóruns, e outras aplicações.


O que se aprende:

  • Competências para desenvolver sites, portais, aplicativos, plataformas, entre outras soluções digitais;
  • Compatibilidades para gerir sites, portais, aplicativos, plataformas, entre outras soluções digitais;
  • Capacidades para manter sites, portais, aplicativos, plataformas, entre outras soluções digitais.

Duração:

Se o seu interesse é cursar a graduação de Sistemas para Internet ele possui uma duração de 3 (três) anos em nossa instituição.
